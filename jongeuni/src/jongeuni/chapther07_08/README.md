8챕터는 교과서를 읽는 느낌이었다. 의존성과 결합도에 대해 확실히 알 수 있었다. (둘의 명확하게 차이를 짚어준 부분이 좋았다.)

9챕터는 팩토리 패턴에 대해 알려준다고 하는데 공감되는 코드가 많이 나올 것 같아서 기대가 된다.
